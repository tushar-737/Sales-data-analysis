# Sales Data Analysis

## Project Overview

This project analyzes sales data using Python and Pandas.

## Features

- Read CSV file
- Explore dataset
- Handle missing values
- Remove duplicates
- Calculate statistics
- Find best-selling product
- Generate sales report

## Installation

```bash
pip install pandas
```

Run

```bash
python sales_analysis.py
```

## Files

- sales_analysis.py
- sales_data.csv
- analysis_report.md
