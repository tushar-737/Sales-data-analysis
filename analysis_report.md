# Sales Data Analysis Report

## Objective

Analyze a sales dataset using Python Pandas.

---

## Dataset Information

- Rows: 100
- Columns: 7

Columns:
- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## Data Cleaning

- Checked missing values
- Filled missing values (if any)
- Removed duplicate records

---

## Metrics Calculated

1. Total Revenue
2. Average Sales
3. Highest Sale
4. Lowest Sale
5. Best Selling Product
6. Revenue by Region

---

## Findings

- Total Revenue: ₹12,365,048
- Best Selling Product: Laptop
- Dataset contained no missing values.

---

## Technologies Used

- Python
- Pandas
- CSV Files

## Conclusion

Pandas makes data analysis simple and efficient. The project demonstrates loading data, cleaning it, computing statistics, and generating business insights.
